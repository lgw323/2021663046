
# 웹 프로그래밍 실습 저장소

이 저장소는 HTML, CSS, JavaScript를 포함한 웹 프로그래밍 실습 과제를 기록하고 관리합니다. 각 과제는 학습 단원별로 정리되어 있으며, 실습 결과물을 직접 확인할 수 있는 링크가 포함되어 있습니다.

- 내용: 오픈 챌린지, 과제별 HTML/CSS/JavaScript 코드
